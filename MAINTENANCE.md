# نصائح صيانة مستقبلية

- راقب حجم مجموعات `AuditLog` و `Notification` وأضف TTL إن بدأت تنمو بسرعة.
- أضف اختبارات تكامل لـ flows الحرجة: إنشاء طلب -> خصم المخزون -> إنشاء فاتورة -> تعيين سائق -> التسليم.
- استخدم Mongo Memory Server في CI لاختبارات أسرع ومعزولة.
- ضع مراقبة للأداء (APM) وتتبّع بطيء الاستعلامات عبر فهارس إضافية عند الحاجة.
